# 🤖 Архитектура Telegram-бота (Snippets)

Добро пожаловать! Здесь представлены базовые фрагменты кода, демонстрирующие мой подход к разработке Telegram-ботов на Python. 

Поскольку исходные коды моих коммерческих проектов находятся под NDA (соглашением о неразглашении), я собрал в этом репозитории безопасные выдержки, которые показывают стиль моего кода, работу с базой данных и архитектуру приложения.

## 🛠 Стек технологий, который я использую
* **Python 3.10+**
* **aiogram 3.x** — современный асинхронный фреймворк для Telegram API.
* **asyncpg** — сверхбыстрый драйвер для работы с PostgreSQL.

## 📂 Описание файлов
* `db_core.py` — пример инициализации структуры базы данных и работы с пулом соединений `asyncpg.Pool` (для удержания нагрузки при большом количестве пользователей).
* `handlers.py` — пример реализации машины состояний (FSM) для пошагового сбора информации от клиента и ее валидации (создание заявки/анкеты).
